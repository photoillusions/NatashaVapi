# Image Organization Guide

## Folder Structure

```
public/
├── images/
│   ├── venues/
│   │   ├── vault-ballroom.jpg        (Vault Ballroom venue image)
│   │   ├── liberty-palace.jpg        (Mae's Liberty Palace image)
│   │   └── banquet-facility.jpg      (Natasha Mae's Banquet Facility image)
│   ├── gallery/
│   │   ├── gallery-01.jpg
│   │   ├── gallery-02.jpg
│   │   └── ...more gallery images
│   └── hero/
│       └── hero-background.jpg       (Hero section background)
```

## How to Add Images

1. **Venue Images**: Save your venue photos to `public/images/venues/` with names:
   - `vault-ballroom.jpg` - The Vault Ballroom
   - `liberty-palace.jpg` - Mae's Liberty Palace  
   - `banquet-facility.jpg` - Natasha Mae's Banquet Facility

2. **Gallery Images**: Save gallery photos to `public/images/gallery/`
   - Name them: `gallery-01.jpg`, `gallery-02.jpg`, etc.

3. **Hero Images**: Save background images to `public/images/hero/`

## Image Format Recommendations

- **Format**: JPG or PNG
- **Venue Images**: Recommended size 1200px wide (aspect ratio 16:9 for proper display)
- **Gallery Images**: Recommended size 800px wide
- **Optimize**: Compress images to reduce load time
